# Libredirect Official Site

- No Javascript, Only HTML & CSS
- No External Dependencies
- Less than 1MB per load
- Fully hosted on Github pages



## Installation

```bash
npm install
```

## Usage

Convert the pages from [Pug](https://pugjs.org) to HTML:
```bash
npm run pug
```
